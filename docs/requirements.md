# 기능 요구사항

## Todo & 일정 관리

- 할 일 추가 / 수정 / 삭제 / 완료 상태 토글
- 마감 기한(Due Date) 및 알람 시간 설정
- 타이머 기능을 통한 작업 집중도 기록 및 캐릭터 레벨업
- 캘린더 뷰 (부가 기능)

## 캐릭터 상호작용 (다마고치 시스템)

| 트리거 | 캐릭터 반응 |
|---|---|
| 할 일 완료 | 춤추는 애니메이션 |
| 할 일 미완료 | 시무룩한 상태 |
| 타이머 작동 중 | 공부/업무 모드로 변신 |
| 할 일 완료 누적 | 친밀도 & 레벨 상승 |

> **구현 팁**: LottieFiles에서 귀여운 동물 JSON 애니메이션을 가져와 `react-lottie`로 제어하면 적은 코드로 캐릭터에 생명력을 부여할 수 있음.

## 스마트 알림 시스템

- **텔레그램 봇**: 마감 알림을 캐릭터 말투로 전송
- **서버 스케줄링**: BullMQ 기반으로 앱 종료 시에도 알림 발송
- **커스텀 알람**: 알람 시점 선택 (10분 전, 정각 등)
