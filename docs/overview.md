# 프로젝트 개요

> "내 할 일을 함께 관리하고 성장하는 다마고치형 데스크톱 메이트"

## 배경

단순 Todo 리스트를 넘어, 기기 간 실시간 동기화와 확실한 외부 알림(텔레그램)을 제공하는 사용자 친화적 시스템. 사용자의 할 일 상태에 따라 실시간으로 반응하는 **캐릭터 기반 상호작용**이 핵심 차별점.

## 핵심 가치

| 가치 | 설명 |
|---|---|
| 데이터 무결성 | 멀티 디바이스 간 실시간 동기화 |
| 알람 정확성 | 앱 종료 시에도 서버 스케줄링으로 알림 발송 |
| 감성적 교감 | 할 일 상태에 반응하는 캐릭터 시스템 |

## 형태

- **데스크톱 앱**: Electron (투명 배경 위젯 모드 지원)
- **모바일 앱**: React Native (예정)
- **블로그 위젯**: 임베드용 웹 위젯 (예정)

## 향후 확장 계획

- 캐릭터 코스튬 상점 (집중 시간 포인트로 구매)
- 트위터(X) API 연동을 통한 공개 다짐 포스팅
- 친구의 캐릭터와 상호작용하는 공유 Todo 리스트
